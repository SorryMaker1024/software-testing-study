## 参考速记

| 概念 | 一句话解释 |
|------|-----------|
| 黑盒测试 | 不看代码，只看输入输出 |
| 白盒测试 | 看代码逻辑，覆盖路径 |
| 冒烟测试 | 上线前快速验证核心功能 |
| 回归测试 | 改了代码后验证旧功能没坏 |
| 等价类 | 把输入分成有效/无效两类，每组只测一个代表值 |
| 边界值 | 刚好等于、刚好大于、刚好小于边界 |
