## 关键语法速记

| 需求 | 写法 |
|------|------|
| 分组统计 | `GROUP BY 字段 HAVING 条件` |
| 多表联查 | `FROM A LEFT JOIN B ON A.id = B.a_id` |
| 子查询 | `WHERE col IN (SELECT ...)` |
| 排序取前N | `ORDER BY col DESC LIMIT N` |
| 去重 | `SELECT DISTINCT col FROM table` |
