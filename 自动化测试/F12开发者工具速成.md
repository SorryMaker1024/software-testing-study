# F12 开发者工具速成

> F12 是测试必备技能，打开浏览器按 F12 就能用。
> 核心只记三个面板：Elements、Console、Network。

---

## 一、怎么打开

| 方式 | 操作 |
|------|------|
| 快捷键 | 按 **F12** |
| 右键 | 页面任意位置右键 → 检查 |
| 菜单 | 浏览器右上角 `⋯` → 更多工具 → 开发者工具 |

打开后在右上角 `⋮` → **Dock side** 可以选底部/右侧/独立窗口显示。

---

## 二、Elements（元素定位）

### 干什么用
查看页面 HTML 结构，找元素属性，写 Selenium 定位表达式。

### 核心操作

**1. 点选元素**
- 点 Elements 面板左上角的箭头图标 📍（或 Ctrl+Shift+C）
- 鼠标移到页面上，自动高亮对应元素
- 点一下，Elements 面板自动展开到对应 HTML 代码

**2. 右键 → Copy**
选中元素后右键，最常用的三个：
```
Copy selector        → CSS 选择器
Copy XPath           → 完整 XPath
Copy JS path         → JS 路径
```

**3. Ctrl+F 搜索**
在 Elements 面板按 Ctrl+F，输入关键词搜索 HTML。

**4. 样式面板**
选中元素后，右边显示所有 CSS 样式。可以看到元素宽高、颜色、边距。

### 测试中怎么用
```
你用 Selenium 定位不到元素 → F12 找到元素
→ 右键 Copy XPath → 贴到代码里 → 搞定
```

---

## 三、Console（控制台）

### 干什么用
看 JS 报错、手动执行 JS 代码调试。

### 核心操作

```javascript
// 直接写 JS 代码，回车就执行
document.title                          // 看页面标题
document.querySelector('.btn')          // 找元素
document.cookie                         // 看 cookie
console.log('hello')                     // 输出日志
```

**红色报错** = 前端 JS 出 bug 了。测试时截图红色报错附到 Bug 报告里。

**筛选**：Console 上方有筛选按钮，可以只看 Error / Warning / Info。

## 四、Network（网络抓包）⭐ 最重要

### 干什么用
看每个 HTTP 请求的完整信息：URL、方法、Headers、Body、响应、耗时。

### 核心操作

1. **先打开 F12，再切到 Network 标签**
2. **刷新页面或操作**，所有请求会被记录
3. **点某一个请求**，右侧出现详细信息：
   - **Headers** → 请求头、响应头
   - **Payload / Request** → 请求体（POST 发了什么数据）
   - **Preview / Response** → 服务器返回的内容（格式化显示）
   - **Timing** → 每个阶段耗时（DNS、连接、等待、下载）

### 常用过滤

| 按钮 | 过滤内容 |
|------|------|
| All | 全部 |
| XHR / Fetch | 只看接口请求（最重要） |
| JS | 只看 JS 文件 |
| CSS | 只看样式 |
| Img | 只看图片 |
| Media | 音视频 |
| Doc | HTML 文档 |

### 测试中怎么用

```
1. 开发说"接口已经返回数据了" → 你点开看 Response 有没有
2. Bug："点了按钮没反应" → 看 Network 有没有发请求出去
   有请求 → 后端问题
   没请求 → 前端问题
3. 性能问题 → 看 Timing，哪个阶段慢
4. Postman 不知道怎么调 → 先在网页操作一遍
   → F12 Network 看实际发的请求长什么样
   → 照抄到 Postman
```

## 五、实战练习（现在做）

打开百度首页，按 F12，做这几个操作：

1. **Elements**：用 📍 箭头选中搜索框，右键 → Copy XPath
2. **Console**：输入 `document.title` 回车，看返回什么
3. **Network**：切到 XHR 过滤，在搜索框随便输入点东西，看有没有网络请求出现
